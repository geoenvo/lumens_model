Algorithm description tes

tes QUES-C adalah modul bagian dari LUMENS (Land Use Planning for Multiple Environmental Services) yang berfungsi untuk melakukan kuantifikasi perubahan cadangan karbon pada skala bentang lahan.

Input parameters

Land use map-first time series

QUES-C melakukan analisa berdasarkan dua peta penutupan lahan seri waktu. Masukan datanya adalah peta tutupan lahan dalam format RASTER. Format data yang bisa digunakan adalah: GeoTiff (.tif), IMAGINE (.img), IDRISI Raster (.rst), GRID (.grd) dan ASCII raster (.asc).

Land use map-second time series

QUES-C melakukan analisa berdasarkan dua peta penutupan lahan seri waktu. Masukan datanya adalah peta tutupan lahan dalam format RASTER. Format data yang bisa digunakan adalah: GeoTiff (.tif), IMAGINE (.img), IDRISI Raster (.rst), GRID (.grd) dan ASCII raster (.asc).

Planning unit map

Peta unit perencanaan dalam format RASTER. Format data yang bisa digunakan adalah: GeoTiff (.tif), IMAGINE (.img), IDRISI Raster (.rst), GRID (.grd) dan ASCII raster (.asc).

Carbon stock lookup table

Tabel cadangan karbon untuk masing-masing tipe tutupan lahan. Format data yang bisa digunakan adalah CSV (.csv), TEXT file (.txt).

Planning unit lookup table

Tabel nama unit perencanaan dalam format RASTER. Format data yang bisa digunakan adalah CSV (.csv), TEXT file (.txt)

Working directory

Lokasi penyimpanan data hasil analisa QUES-C.

Analysis location

Nama lokasi geografis dari daerah yang akan dianalisa

Epoch of the first time series

Waktu pengambilan data peta tutupan lahan seri waktu pertama.

Epoch of the second time series

Waktu pengambilan data peta tutupan lahan seri waktu pertama.

Outputs

Carbon 1

Carbon 2

Emission

Sequestration

Emission by zone

Summary

Plot
